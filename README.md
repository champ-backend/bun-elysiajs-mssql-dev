# Elysia with Bun runtime

## Getting Started

To get started with this template, simply paste this command into your terminal:

```bash
bun create elysia ./elysia-example
```

## Localhost

To start the localhost server run:

```bash
bun run localhost
```

## Development

To start the development server run:

```bash
bun run development
```

## Production

To start the production server run:

```bash
bun run production
```

Open http://localhost:42001/ with your browser to see the result.
